Planeta Alien

A exploração do planeta Alien necessita não apenas de TAGS. É importante explorar propriedades. As propriedades "value" e "innerText" são as mais utilizadas. A propriedade "value" obtém ou altera o conteúdo de um campo de formulário HTML. Portanto, para obter o nome do usuário informado,é preciso utilizar essa propriedade. Já a propriedade innerText, serve para alterar ou obter o conteúdo de elementos de texto do documento identificados no código HTML. É possível, portanto, alterar o texto de qualquer parágrafo ou texto de cabeçalho em uma página web Utilizando essa propriedade. Há também a propriedade innerHTML, semelhantemente, a innerText, porém, renderiza os códigos HTML existentes no seu conteúdo.
